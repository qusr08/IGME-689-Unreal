# Controls
- W : Accelerate
- S : Brake/Reverse
- A/D : Steer Left/Right
- C : Toggle Camera Perspective
- Move Mouse : Pan Camera Around Car

# How To Play
Move as fast as you can around the track and try and beat the posted time! You must hit every checkpoint along the track in order to finish the race.

# Other Features
Change the "Active Feature Index" in the Details Pane for the Feature Layer Query object in the scene. This will automatically select that F1 track index and allow you to race on it.
